int *  internal_index,
int *  internal_buf_len 
)

Moves forwards 1 entry in the user's command history.

Parameters
internal_bufThe buffer managed by low-level read operations containing user input from the terminal. Contents will be overwritten with the next entry in the user's command history.
internal_indexA pointer to the position of the cursor, managed by low-level read operations. The cursor position will be adjusted to point to the end of the line.
internal_buf_lenA pointer to the length of the buffer containing user input to the terminal. Will be adjusted to contain the length of the history entry being written to the buffer.
